# PRD — 아나빠나사띠(Ānāpānasati) 호흡명상 웹서비스

> **프로젝트명(가칭):** 숨결의 길 (Ānāpānasati Path)
> **작성일:** 2026-07-22
> **작성자:** 김민영 (소울매트 / 소울라이즈)
> **버전:** v1.0 (MVP 기획)

---

## 1. 문제 정의 (Problem)

### 배경
- 부처가 열반에 들기 전 가르친 **아나빠나사띠(호흡 알아차림 명상)** 는 불교 명상의 핵심이자, 현대 마음챙김(mindfulness)의 원류다.
- 그러나 대부분의 명상 앱은 **"눈 감고 따라하세요"** 식의 단순 타이머·음원 재생에 그친다. 아나빠나사띠가 본래 담고 있는 **16단계(4념처 × 4단계)의 체계적 수행 구조**를 배우고 실천하며 기록할 수 있는 도구는 드물다.

### 해결하려는 문제
1. **입문자**는 아나빠나사띠가 무엇이고 어떤 단계로 이루어지는지 배울 곳이 마땅치 않다.
2. **수련생**은 매일의 호흡 명상을 안내받고, 수행 기록을 꾸준히 남기며 성장을 되돌아볼 도구가 필요하다.
3. 두 니즈를 **하나의 흐름(배우기 → 수행하기 → 기록하기)** 으로 이어주는 서비스가 없다.

### 한 문장 정의
> "아나빠나사띠 16단계를 **배우고(Learn)**, 호흡 가이드로 **수행하고(Practice)**, 수행을 **기록(Journal)** 하는 하나의 고요한 공간."

## 2. 타겟 유저 (Target Users)

| 페르소나 | 설명 | 핵심 니즈 |
|----------|------|-----------|
| **입문자 '지현'** (28세) | 명상을 시작하고 싶지만 방법을 모름. 스트레스·불면 | 쉬운 설명, 부담 없는 짧은 호흡 가이드 |
| **수련생 '도윤'** (45세) | 요가·명상 3년차. 꾸준한 수행 습관화를 원함 | 단계별 심화 학습, 세션 기록, 성장 추적 |
| **커뮤니티 회원** | 소울라이즈 동아리 등 함께 수행하는 사람들 | 자기 기록 관리, 향후 공유 기능 확장 여지 |

**공통 가치:** 광고·소음 없는 고요한 환경에서, 정통 가르침에 기반한 호흡 명상을 신뢰하며 실천.

## 3. 핵심 기능 (MVP Features)

### 반드시 있어야 할 기능 (Must-have)

| # | 기능 | 설명 | 퀘스트 요건 매핑 |
|---|------|------|------------------|
| F1 | **16단계 학습** | 4념처(몸·느낌·마음·법) × 4단계를 카드/목록으로 열람, 각 단계 상세 설명 | 프론트: 목록+상세 화면 |
| F2 | **호흡 가이드 (인터랙티브 타이머)** | 들숨-멈춤-날숨-멈춤을 시각 애니메이션으로 안내. 시간·패턴 설정 가능 | 프론트: 입력→결과 흐름 |
| F3 | **수행일지 기록 (Create)** | 명상 세션 종료 후 시간·수행단계·소감·집중도를 저장 | 백엔드 CRUD(C) + DB |
| F4 | **수행일지 조회 (Read)** | 내 지난 세션들을 리스트로 조회, 누적 통계(총 시간·연속일수) | 백엔드 CRUD(R) + DB |
| F5 | **일지 수정/삭제 (Update/Delete)** | 기록한 세션 소감 수정, 잘못된 기록 삭제 | 백엔드 CRUD(U/D) |
| F6 | **이메일 로그인/회원가입** | Supabase Auth로 유저별 데이터 분리 | 확장(가산점) + DB RLS |

### 있으면 좋은 기능 (Nice-to-have, 시간 여유 시)
- 명상 배경음(앰비언트 사운드) 토글
- 수행 통계 시각화(주간 캘린더 히트맵)
- **AI 수행 코칭** — Claude API로 일지 소감에 대한 짧은 격려·조언 (가산점: AI 기능)
- 반응형 모바일 최적화 (필수에 준함)

### 명시적 범위 제외 (Out of Scope for MVP)
- 소셜 공유·팔로우, 댓글
- 결제·유료 구독
- 오디오 음원 업로드/스트리밍(대용량)
- 다국어(우선 한국어)

## 5. 데이터 구조 (Data Model)

### DB: Supabase (PostgreSQL)

#### `profiles` (사용자 프로필 — auth.users와 1:1)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid (PK, = auth.users.id) | 사용자 ID |
| display_name | text | 표시 이름 |
| created_at | timestamptz | 가입일 |

#### `sessions` (수행일지 — 핵심 테이블)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid (PK) | 세션 ID |
| user_id | uuid (FK→auth.users) | 작성자 |
| duration_sec | int | 수행 시간(초) |
| stage | int (1~16) | 수행한 아나빠나사띠 단계 |
| breath_pattern | text | 호흡 패턴 (예: "4-4-6-2") |
| focus_score | int (1~5) | 자기평가 집중도 |
| note | text | 소감/메모 |
| ai_feedback | text (nullable) | AI 코칭 피드백 (선택) |
| created_at | timestamptz | 기록 시각 |

#### `stages` (16단계 학습 콘텐츠 — 정적 데이터)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | int (PK, 1~16) | 단계 번호 |
| tetrad | int (1~4) | 소속 념처(몸/느낌/마음/법) |
| title_ko | text | 한글 제목 |
| title_pali | text | 빠알리어 원제 |
| description | text | 해설 |
| practice_tip | text | 수행 팁 |

> `stages`는 초기 시드 데이터로 고정(16행). 프론트에 정적 포함하거나 DB 조회 둘 다 가능 — MVP에선 DB 조회로 "데이터 불러오기 흐름" 시연.

### 보안 (RLS - Row Level Security)
- `sessions`, `profiles`: 본인 데이터만 조회/수정/삭제 가능 (`auth.uid() = user_id`)
- `stages`: 전체 공개 읽기 전용

## 6. 기술 스택 (Tech Stack)

| 레이어 | 기술 | 비고 |
|--------|------|------|
| 프론트엔드 | **React + Vite** | SPA, React Router |
| 스타일 | CSS (또는 Tailwind) | 고요한 Zen 톤, 여백·뉴트럴 |
| 백엔드/API | **Supabase** | PostgREST 자동 API(CRUD), Auth |
| DB | **Supabase (PostgreSQL)** | RLS로 유저별 격리 |
| 인증 | **Supabase Auth** | 이메일 로그인 |
| 배포 | **GitHub Pages** | Actions로 자동 빌드·배포 (SPA 라우팅 대응) |
| (선택) AI | **Claude API** | 일지 소감 코칭 |

> 백엔드 API는 Supabase의 자동 생성 REST/JS SDK를 사용해 CRUD 구현. 퀘스트의 "요청 처리 API 서버" 요건을 Supabase(BaaS)로 충족.

## 7. 디자인 방향 (Design Direction)

- **컨셉:** 고요한 모다른 선(Zen). 명상 자체가 되는 화면.
- **여백:** 넉넉한 화이트스페이스, 요소 최소화.
- **색:** 부드러운 뉴트럴 — 미색/모래빛/차분한 세이지·클레이 포인트.
- **타이포:** 가독성 높은 세리프/산세리프 혼용, 큰 자간·행간.
- **모션:** 호흡 애니메이션은 느리고 부드럽게(ease-in-out), 원의 확장·수축.
- **다크모드:** 향후 확장(밤 명상용).

## 부록 A. 아나빠나사띠 16단계 (콘텐츠 시드 원본)

### 제1 념처 — 몸(身, Kāya)
1. 긴 호흡을 길게 알아차림
2. 짧은 호흡을 짧게 알아차림
3. 온몸(호흡 전체)을 경험하며 호흡
4. 몸의 작용(호흡)을 고요히 하며 호흡

### 제2 념처 — 느낌(受, Vedanā)
5. 기쁨(pīti)을 경험하며 호흡
6. 행복(sukha)을 경험하며 호흡
7. 마음의 작용(느낌·지각)을 경험하며 호흡
8. 마음의 작용을 고요히 하며 호흡

### 제3 념처 — 마음(心, Citta)
9. 마음을 경험하며 호흡
10. 마음을 기쁘게 하며 호흡
11. 마음을 집중(삼매)하며 호흡
12. 마음을 해방하며 호흡

### 제4 념처 — 법(法, Dhamma)
13. 무상(無常)을 관찰하며 호흡
14. 사라짐(離慾)을 관찰하며 호흡
15. 소멸(滅)을 관찰하며 호흡
16. 놓아버림(放棄)을 관찰하며 호흡
